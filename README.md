🍔 Cravyo - Food Delivery System

🚀 A full-stack food delivery web application built using HTML, CSS, JavaScript, Python (Flask), and SQLite.

🌟 Features

🛒 Add to cart functionality

🔐 User login/Sign system

🍽️ Browse food items

💾 Data stored using SQLite database

⚡ Fast and responsive UI

🛠️ Tech Stack

Frontend: HTML, CSS, JavaScript

Backend: Python (Flask)

Database: SQLite

▶️ How to Run Locally

Clone the repository

git clone https://github.com/sidcloudio/Cravyo-Food-Delivery-System.git

Go to project folder

cd Cravyo-Food-Delivery-System

Install dependencies

pip install flask

Run the server

python app.py

Open in browser

http://127.0.0.1:3000/index.html?vscode-livepreview=true
